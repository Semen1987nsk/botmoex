# MOEX Linear Regression Bot

Telegram бот для отслеживания пересечения каналов линейной регрессии на инструментах Московской Биржи (Акции, Фьючерсы, Облигации).

## Функционал
- Мониторинг всех инструментов: Акции (TQBR), Фьючерсы (RFUD), Облигации (TQOB).
- Таймфрейм: 15 минут (ресемплинг из минутных данных).
- Стратегия: Линейная регрессия (длина 200) +/- 4 стандартных отклонения.
- Уведомление при выходе цены закрытия за границы канала.

## Установка

1. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` и добавьте токен:
   ```bash
   cp .env.example .env
   ```

## Запуск

```bash
python main.py
```

Команды:
- `/start` - Приветствие
- `/scan` - Запуск цикла сканирования
